HTML Text Formatting

Understanding HTML Formatting Tags

This bold text is used to emphasize words, while important text highlights something critical.

This italic text and emphasized text can be used for titles or highlights.

You can also highlight text for attention.

Smaller text can be shown using the small tag.

Strikethrough text is represented with deleted text and underlined text with inserted text.

Chemical formulas like H2O and mathematical expressions like x2 are formatted using subscript and superscript.
